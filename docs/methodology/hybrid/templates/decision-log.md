# Decision Log (Hybrid)

## Purpose
Capture cross-phase decisions, rationale, and impacts to ensure transparency in hybrid delivery.

## Audience
Project Manager, Steering Committee, Delivery Leads.

## Decision Log Entries
| Decision ID | Date | Decision | Rationale | Phase Impact | Sprint Impact | Owner | Status | Follow-up Actions |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |  |  |

## Completion Checklist
- [ ] Decisions include rationale and impact assessment.
- [ ] Owners and follow-up actions assigned.
- [ ] Decision status reviewed at governance cadence.

## Acceptance Criteria
- Decision log reflects approved, traceable decisions.
- Stakeholders can identify impacts across phases and sprints.
