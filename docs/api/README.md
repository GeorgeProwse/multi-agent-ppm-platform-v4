# API Documentation Index

This directory contains API contracts for the Multi-Agent PPM platform.

## Core APIs
- **API Gateway**
  - OpenAPI docs served at `/api/docs` and `/api/redoc`.
- **Orchestration Service**
  - `docs/api/orchestration-openapi.yaml`
- **Auth & identity**
  - `docs/api/auth.md`
- **Event contracts**
  - `docs/api/event-contracts.md`
- **Webhooks**
  - `docs/api/webhooks.md`

## Usage
- Use the OpenAPI specs for client generation.
- Validate changes with `scripts/validate-schemas.py` and contract tests.
